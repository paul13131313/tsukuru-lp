STEP 1 / 6

配信設定をはじめましょう

これから、メール配信に必要な設定をご案内します。

Resend（レゼンド）という無料のメール配信サービスに
アカウントを作っていただくだけです。

難しい操作はありません。
画面の指示にそって進めれば大丈夫です。

⏱ 所要時間：約5分
STEP 2 / 6

Resendのサイトを開きます

下のボタンを押すと、Resendのサイトが
新しいタブで開きます。

このページは閉じずに、そのまま残しておいてください。
あとでこのページに戻ってきます。

⏱ このステップ：約30秒
Resendを開く →
ページが英語で表示される
→ 問題ありません。そのまま進めてください。
ボタンを押しても何も起きない
→ ブラウザがポップアップをブロックしている場合があります。画面上部に「ポップアップがブロックされました」という表示が出ていたら「許可」を押してください。
STEP 3 / 6

Resendに登録する

Resendのページで、以下を入力してください。

  • 1 メールアドレス（普段使っているもの）
  • 2 パスワード（新しく決めてください）
  • 3 入力したら 「Sign Up」ボタンを押す
  • 4 確認メールが届いたらリンクをクリック
⏱ このステップ：約2分
「Sign Up」ボタンが見つからない
→ ページの中央、またはGoogleアカウントやGitHubアカウントでの登録ボタンの下にあります。
確認メールが届かない
→ 迷惑メールフォルダを確認してください。「Resend」という送信元からメールが届きます。5分待っても届かなければ、もう一度「Sign Up」からやり直してください。
すでにResendのアカウントを持っている
→ そのままログインして次のステップへ進んでください。
STEP 4 / 6

API Keysページを開く

ログイン後の画面で、
左側のメニューを見てください。

  • 1 「API Keys」という文字をクリック
  • 2 右上の 「+ Create API Key」 ボタンを押す
⏱ このステップ：約30秒
左メニューが見当たらない
→ 画面左上の「≡」マーク（三本線）を押してください。メニューが開きます。スマートフォンで見ている場合は特にこの操作が必要です。
「API Keys」が見つからない
→ メニューの中に鍵のアイコンと一緒に「API Keys」と書かれています。見つからない場合は、Resendのトップページ左上のメニューを開いて探してください。
STEP 5 / 6

APIキーを作成する

「Create API Key」を押した後の画面で、
以下のように設定してください。

  • 1 Name（名前）に tsukuru と入力
  • 2 Permissionは Full Access を選択
  • 3 「Add」ボタンを押す

画面に 「re_」で始まる長い文字列が表示されたら成功です！

⚠️ 重要：この画面を閉じると、このキーは二度と表示されません。
まだ閉じずに、次のステップで入力欄に貼り付けてください。
うっかり画面を閉じてしまった
→ 大丈夫です。もう一度「API Keys」ページに戻り、「+ Create API Key」から新しいキーを作り直せます。
「Permission」の選択肢が分からない
→ 「Full Access」と「Sending Access」の2つがあります。Full Accessを選んでください。
STEP 6 / 6

APIキーを貼り付ける

先ほどの 「re_」で始まる文字列をコピーして、
下の入力欄に貼り付けてください。

コピーのしかた
・Resend画面上のコピーボタン（四角が2つ重なったアイコン）をクリック
・または、文字列を選択して右クリック →「コピー」

貼り付けのしかた
・入力欄をクリック → 右クリック →「貼り付け」
・または Cmd + V（Mac）/ Ctrl + V（Windows）
メールアドレスを正しく入力してください。
re_ で始まる文字列です
APIキーは re_ で始まる文字列を入力してください。
メールの差出人名になります
