配信設定をはじめましょう

メール配信にはResendを使用します。
APIキーを登録いただければ、あとは私たちが設定を進めます。

Resend APIキーの取得方法
  1. Resendに無料登録
  2. ダッシュボードの「API Keys」を開く
  3. 「Create API Key」で新しいキーを発行
  4. キーをコピーして下の入力欄に貼り付け
re_ で始まる文字列です
